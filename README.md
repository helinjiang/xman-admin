# xman-admin
这是一个后台管理系统，基于 Thinkjs+React+Redux+MySQL+Ant.design 开发而成。

> 这是一个带有实验性的系统，初衷是为了学习新的技术，因此请谨慎用于生产环境中。

## 使用

``` bash
// 在一个窗口下执行如下命令，进行 React 代码的实时编译
npm run webpack

// 另一个窗口下执行如下命令，来启动服务。
npm start
```

首页地址：`http://localhost:8360/`。
管理页面地址：`http://localhost:8360/admin`，登录的用户名为 `admin`
